# SQL Injection Detection Tools Comparison
Different SQLi detections tools have different injection techniques, target database types and interfaces(GUI vs CLI). 
A comprehensive comparison of those tools helps build a reproducible benchmark that can systematically evalutate detection tools.

## Target Vulnerable Applications
- OWASP Juice Shop
- OWASP WebGoat
- DVWA
- bWAPP

## SQL Injection Dedectors
- SQLmap
- JSQL Injection
- DSSS
- Blisqy
- BBQSQL
